# Pet project "Hangman"
Консольная реализация игры "Виселица", написанная на Java версии 17.0.18 в процедурном стиле. Задача пользователя угадать загаданное слово по буквам за 6 попыток
### Функционал
- Выбор рандомного слова из словаря 
- Ввод буквы пользователем
- Отображение текущего состояния слова и виселицы
- Подсчет ошибок
- Завершение игры при победе или поражении
### Как запустить
1. Клонировать репозиторий
 ```bash
git clone https://github.com/ppixiss/hangman.git
cd hangman
 ```
2. Скомпилировать исходный код
 ```bash
javac src/*.java
 ```

3. Запустить программу
```bash
java -cp src Hangman
```
